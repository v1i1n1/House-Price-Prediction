# 🏡 House Price Prediction using Machine Learning

A Machine Learning web application that predicts the selling price of a house based on various house features. The project is built using **Python, Flask, Scikit-learn, HTML, and CSS**.

---

## 📌 Project Overview

This project uses a **Linear Regression Machine Learning model** trained on the Kaggle House Prices dataset. Users can enter house details through a web interface, and the application predicts the estimated selling price instantly.

---

## 🚀 Features

- Predict house prices using Machine Learning
- User-friendly web interface built with Flask
- Real-time prediction
- Responsive and clean UI
- Input validation using HTML
- Clear button to reset all fields
- Displays predicted price instantly

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/v1i1n1/House-Price-Prediction.git
```

### Navigate to the project

```bash
cd House-Price-Prediction
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open your browser

```
http://127.0.0.1:5000
```
